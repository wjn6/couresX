# 每次更新时，注意需要进行 [增量更新]
即，更新步骤：
clone开源源码 → 压缩为压缩包并上传到服务器根目录 → 解压后访问网站域名 → 选择'增量更新' → 搞定
*注意：若安装程序报错 [网络异常，请重试]，请将confing文件夹下的所有文件赋予正确的权限所有者(www)
